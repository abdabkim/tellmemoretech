The Compiler Revolution

Traditional frameworks like React and Vue do most of their work in the browser. They ship a runtime library that manages virtual DOMs, handles reactivity, and updates the UI. Svelte takes a radically different approach ‚Äì it moves all this work to build time.

Traditional Framework (React)
// React Component
function Counter() {
  const [count, setCount] = useState(0);
  return (
    <button onClick={() => setCount(count + 1)}>
      Count: {count}
    </button>
  );
}

// Requires React runtime (~45KB gzipped)
Svelte Component
<script>
  let count = 0;
</script>

<button on:click={() => count += 1}>
  Count: {count}
</button>

// Compiles to vanilla JS (~2KB for this component)
